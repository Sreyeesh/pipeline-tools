Workflow Demo

A quick path from empty project to tasks and versions

# 1) Launch interactive (pick project/app)
pipely

# 2) Create/show (first time)
pipely shows create DMO "Demo" animation_short
pipely shows use -c DMO

# 3) Shot + tasks
pipely shots add SH010 First shot
pipely tasks add DMO_SH010 Layout
pipely tasks add DMO_SH010 Blocking
pipely tasks status DMO_SH010 Layout in_progress

# 4) Asset + tag
pipely assets add -t CH -n Hero
pipely assets tag DMO_CH_Hero hero

# 5) Versioning
pipely versions new DMO_SH010 anim
pipely versions latest --shot DMO_SH010 --kind anim

# 6) Check progress
pipely shots list DMO
pipely tasks list DMO_SH010
pipely assets list -c DMO
pipely versions list --shot DMO_SH010

Artist Workflow (DMO example)

Copy/paste inside the interactive prompt‚Äîno file browsing needed.

# Create and select the show
shows create DMO "Demo" animation_short
shows use -c DMO

# Create the animation bible in 01_ADMIN (Pipely-only)
admin create --name DMO_animation_bible.md --content "Start your bible here"
# Or use the bundled template (if available)
admin template -t animation_bible --name DMO_animation_bible.md
admin files --open DMO_animation_bible.md   # open from Pipely

# Shots (one per beat)
shots add SH010 First beat
shots add SH020 Second beat
shots add SH030 Third beat

# Characters
assets add -t CH -n Hero
assets add -t CH -n Supporting_A
assets add -t CH -n Supporting_B

# Environment & props (use PR for FX beats)
assets add -t ENV -n Main_Set
assets add -t ENV -n Set_Dressing_A
assets add -t ENV -n Set_Dressing_B
assets add -t PR  -n Hero_Prop_A
assets add -t PR  -n FX_Placeholder_A   # FX placeholder using PR

# Tasks (design/layout)
tasks add DMO_CH_Hero Design
tasks add DMO_CH_Supporting_A Design
tasks add DMO_ENV_Main_Set Design
tasks add DMO_SH010 Layout
tasks add DMO_SH020 Layout
tasks add DMO_SH030 Layout

# Workfiles (kept in 05_WORK) and open in Krita
workfiles add DMO_CH_Hero --kind krita --open
workfiles add DMO_SH010 --kind krita

# Versions
versions new DMO_CH_Hero design
versions new DMO_SH010 layout

# Check progress
shots list DMO
tasks list DMO_CH_Hero
tasks list DMO_SH010
assets list -c DMO
versions list --shot DMO_SH010

# Open a DCC against your project context
open krita --project AN_DMO_Demo

pipely

pipely create --interactive
