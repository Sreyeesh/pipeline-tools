Get Started in Seconds

From zero to production-ready pipeline in three simple steps

# Install Pipely (requires Python 3.8+)
pipx install https://github.com/Sreyeesh/pipely/releases/latest/download/pipely-0.1.8-py3-none-any.whl

# Launch interactive mode
pipely

# Or create a project directly
pipely create --interactive

# That's it! Your production pipeline is ready üéâ
